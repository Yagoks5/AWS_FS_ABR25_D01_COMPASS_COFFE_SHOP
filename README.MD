# Coffee Shop

## 📝 Description

**Coffee Shop** is a website for a premium coffee shop that offers an elegant and modern experience. The site features a sleek interface showcasing various coffee types, information about the café, customer reviews, and a subscription system for special offers.

---

## ✨ Features

- **Menu Display**: Full list of available coffees with category filters
- **Product Cards**: Visual display of featured coffees
- **Testimonials Carousel**: Navigate through customer feedback
- **Newsletter System**: Subscription form to receive updates and promotions
- **Item Filtering**: Filter products by category

---

## 🛠️ Technologies Used

- **HTML5**: Content structure
- **CSS3**: Styling and responsive layout
- **JavaScript**: Interactivity and dynamic functionality
- **Fetch API**: Loads dynamic data from JSON files
